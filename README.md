# Candida infections in Tolima (2014–2024) – Reproducible materials

This repository contains R scripts, Excel workbooks, and figure/table outputs supporting the manuscript on *Candida* infections in Tolima, Colombia (2014–2024).

Individual-level clinical and laboratory data are not publicly available due to ethical and legal restrictions. Only aggregated outputs and templates are provided.

## Contents
- `figure1_trend/`, `figure4_PUBLIC.xlsx/`, `figure2_flowchartppt/`: figure sources/exports
- `figure3_age_repo/`, `figure5_mca_repo/`: R workflows for figures
- `table2_models_repo/`, `table3_models/`, `table1_models/`: tables and models
